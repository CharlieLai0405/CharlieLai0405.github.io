Build Chatbots with Amazon Bedrock

Amazon Bedrock

How it works

Prerequisites

  1. Download PDF Guide from Guidelines
  2. 在 Search 找到 S3 去 create 一個 create-customer-resources-kb-bucket
  3. 接著在 Search 找到 Bedrock，選擇 Amazon Bedrock
  4. 在側邊欄位選擇 Model access

  1. 選擇 Enable specific models 來選擇想要用的模型

Knowledge base for Amazon Bedrock

Create Knowledge base

  1. 一樣在 Amazon Bedrock 的側欄選擇 Knowledge bases

  1. 根據需求選擇格式，可選 Knowledge Base with vector store

  1. 參考範例建立 Knowledge Base

  1. 建立完後可上傳資料，記得 Sync(同步)

Test Knowledge base

  1. 同步完可先關掉 Generate responses

  1. Amazon Bedrock 會只傳回相關資料來源中的文字。

Create an Agent

  1. 側欄選擇 Agents → Create Agent

  • 若需多個 Agent 並行，可勾選 Enable multi-agent collaboration
