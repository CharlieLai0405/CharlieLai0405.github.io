Build Realtime Voice Agent on Amazon Bedrock

Application Architecture

透過 CloudFront 存取 Astra 的 Playground 和後端服務。Astra 呼叫以下服務：

  • Amazon Transcribe：語音轉文字
  • Amazon Bedrock：用於對話與翻譯的 Foundation Model
  • Amazon Polly：將文字轉為語音
  • Amazon SageMaker：支援自架 LLM 與 TTS 模型
