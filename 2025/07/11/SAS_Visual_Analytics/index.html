SAS Visual Analytics 應用指南

平台：SAS Viya

SAS Visual Analytics (VA) 是什麼？

SAS Visual Analytics 是架設於 SAS Viya 平台上的應用程式，具有以下特色：

資料探勘與分析

  • 高效能分析引擎，支援大量資料處理
  • 挖掘關聯與趨勢，做為決策依據

報表建立與分享

  • 互動式報表建立工具
  • 團隊共享報表、即時檢視分析結果

Visual Analytics 與 CAS 架構

四大組件與流程

  1. Client（使用者端）：建立報表與視覺化操作
  2. Visual Analytics（VA 應用）：將需求轉為 CAS Action
  3. CAS（分析引擎）：平行運算、資料處理
  4. 資料回傳：CAS 回傳結果供 VA 呈現

所有資料處理皆透過 CAS 執行，呈現流程：分析 → 彙整 → 視覺化

自動解釋功能（Auto Explain）


什麼是「自動解釋物件」？

  • 自動分析目標變數受哪些變數影響最大
  • 提供排序清單與視覺化圖表，支援建模與解釋
